# Smart Home Management using SNMP

A lightweight Smart Home management platform implementing an SNMP-inspired architecture for monitoring and controlling connected IoT devices.

The project simulates a smart home environment where a central manager communicates with multiple agents responsible for controlling lighting, air conditioning and environmental monitoring.

---

## Features

- Smart Home Manager
- SNMP-inspired communication protocol
- Device monitoring
- Lighting control
- Air conditioning control
- Environmental monitoring
- Graphical User Interface (GUI)
- Real-time communication using UDP sockets

---

## Technologies

- Python
- Socket Programming
- UDP
- Tkinter
- Networking
- SNMP Concepts

---

## Objective

The objective of this project was to understand network management concepts by implementing a simplified SNMP architecture capable of monitoring and controlling multiple simulated IoT devices.
